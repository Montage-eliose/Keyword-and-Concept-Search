NNAAMMEE
       byobu-select-backend - select your default Byobu backend window manager


SSYYNNOOPPSSIISS
       bbyyoobbuu--sseelleecctt--bbaacckkeenndd [screen|tmux]


DDEESSCCRRIIPPTTIIOONN
       bbyyoobbuu--sseelleecctt--bbaacckkeenndd  is  an application that lists the available Byobu
       backends and allows you to select your default.

       You can select one of either ssccrreeeenn(1) or ttmmuuxx(1) on the command  line,
       or  giving  no  command  line parameters, the utility will run interac-
       tively.

       This utility will only affect which backend is  used  by  default  when
       simply  running,  bbyyoobbuu(1).   Note  that at any time, you can force the
       backend of a particular session by running the helpers, bbyyoobbuu--ssccrreeeenn(1)
       or bbyyoobbuu--ttmmuuxx(1).
