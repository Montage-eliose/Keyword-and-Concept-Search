       grub-render-label - generate a .disk_label for Apple Macs.

       Render Apple .disk_label.

       --bb, ----bbggccoolloorr=_C_O_L_O_R
              use COLOR for background

       --cc, ----ccoolloorr=_C_O_L_O_R
              use COLOR for text

       --ff, ----ffoonntt=_F_I_L_E
              use FILE as font (PF2).

       --ii, ----iinnppuutt=_F_I_L_E
              read text from FILE.

       --oo, ----oouuttppuutt=_F_I_L_E
              set output filename. Default is STDOUT

       --tt, ----tteexxtt=_S_T_R_I_N_G
              set the label to render

       --vv, ----vveerrbboossee
              print verbose messages.

       -?, ----hheellpp
              give this help list

       ----uussaaggee
              give a short usage message

       --VV, ----vveerrssiioonn
              print program version

       Mandatory  or  optional arguments to long options are also mandatory or
       optional for any corresponding short options.
