       ppyyddoocc33..44  _n_a_m_e  Show  text documentation on something.  _n_a_m_e may be the
       name of a Python keyword, topic, function, module,  or  package,  or  a
       dotted  reference to a class or function within a module or module in a
       package.  If _n_a_m_e contains a '/', it is used as the path  to  a  Python
       source file to document. If name is 'keywords', 'topics', or 'modules',
       a listing of these things is displayed.

       ppyyddoocc33..44 --kk _k_e_y_w_o_r_d Search for a keyword in the synopsis lines  of  all
       available modules.

       ppyyddoocc33..44  --pp  _p_o_r_t  Start an HTTP server on the given port on the local
       machine.

       ppyyddoocc33..44 --gg Pop up a graphical interface for finding and serving  docu-
       mentation.

       ppyyddoocc33..44  --ww  _n_a_m_e _[_._._._]  Write out the HTML documentation for a module
       to a file in the current directory.  If _n_a_m_e  contains  a  '/',  it  is
       treated  as a filename; if it names a directory, documentation is writ-
       ten for all the contents.
