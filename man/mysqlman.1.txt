       mysqlman - default man page for mysql

DDEESSCCRRIIPPTTIIOONN
       Certain executables distributed with the MySQL database management sys-
       tem do not have specific man pages.

SSEEEE AALLSSOO
       In most cases, you can run the executable from the command line with  a
       "--help"  argument to display a brief summary of the executable's argu-
       ments and function.  For more information about MySQL, please refer  to
       the  MySQL reference manual, which may already be installed locally and
       which is also available online at http://dev.mysql.com/doc/mysql/en

BBUUGGSS
       Please refer to http://bugs.mysql.com/ to report bugs.
