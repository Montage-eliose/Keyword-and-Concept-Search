       NNggiinnxx  ("engine  X") is a high-performance web and reverse proxy server
       created by Igor Sysoev. It can be used both as a standalone web  server
       and as a proxy to reduce the load on back-end HTTP or mail servers.

       A summary of options is included below:

       --??,--hh  Show this help.

       --vv     Show version and exit.

       --VV     Show version and configure options then exit.

       --ss _s_i_g_n_a_l
              Send signal to a master process: stop, quit, reopen, reload.

       --pp _p_r_e_f_i_x
              Set prefix path.

       --gg _d_i_r_e_c_t_i_v_e_s

       --cc <_c_o_n_f_i_g_u_r_a_t_i_o_n _f_i_l_e>
              Specifies a particular configuration file for nginx to load.

       --tt     Tests nginx configuration and exit.
