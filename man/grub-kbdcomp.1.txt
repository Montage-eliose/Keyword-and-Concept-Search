       grub-kbdcomp  processes  a  X keyboard layout description in kkeeyymmaappss(5)
       format into a format that can be used by GRUB's kkeeyymmaapp command.

       Make GRUB keyboard layout file.

       --hh, ----hheellpp
              print this message and exit

       --vv, ----vveerrssiioonn
              print the version information and exit

       --oo, ----oouuttppuutt=_F_I_L_E
              save output in FILE [required]

       grub-kbdcomp generates a keyboard layout for GRUB using ckbcomp
